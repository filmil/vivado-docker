# Instructions for Gemini

When working in this repository, please adhere to the following guidelines:

1.  **Commit Messages:** Always use the [Conventional Commits](https://www.conventionalcommits.org/)
    specification for writing your commit messages
    (e.g., `feat:`, `fix:`, `docs:`, `chore:`, etc.).
2.  **General Instructions:** Before making any changes or starting a task,
    read the `README.md` file in the root directory for general
    instructions and context about the project.
3.  **Markdown Formatting:** Each edited markdown file should be
    reformatted to an 80-column layout.
